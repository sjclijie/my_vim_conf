# my_vim_conf
我的vim配置

需要先安装ctags

====

默认leader键: ';'

集成插件:
    - NERDTree         快捷键<F3>
    - TarBar           快捷键<F4>
    - miniBufExpl      快捷键<F5>
    - EasyMotion       <leader><leader>f
    - NERDCommenter    <leader>cc 注释, <leader>cu 取消注释
    - VisIncr
    - vim-powerline 
    - vim-indent-guides
    - emmet
    - AutoComplPop
